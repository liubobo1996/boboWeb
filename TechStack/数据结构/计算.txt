计算数组的插入或者删除

和 N 无关的数, 都是常数, 可以省略掉

1. 数组有 N 个元素

在第 k 个位置插入一个元素
后面有 (N - k) 个元素
假设一个元素往后移动一个位置, 耗时 x

总时间 = x * (N - k)
      = xN - xK
算时间复杂的时候, 可以把和 N 相乘的常数当成 1
O(N) = N - 1
可以把这个相加的常数 1 忽略
所有最终的结果就是 O(N) = N
写成 O(N)
时间复杂度就是 O(N)


2. 链表有 N 个元素
在第 k 个位置插入一个元素
假设走一步用时 x

走到第 K 个位置, 用时 x * k
假设插入元素用时 y

x * k + y

所有时间复杂度就是 O(xk + y) 也就是 O(1)
